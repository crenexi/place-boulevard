# mistan-city

This React-based project is a small visual tracker for my hobby project **Mistan City**. Obsessed with [*Cities:Skylines*](https://en.wikipedia.org/wiki/Cities:_Skylines) since the day it launched, I'm building my grandest of cities one system and district at a time. This app lists the commissions, departments, and agencies of my fictional, coastal California city.
